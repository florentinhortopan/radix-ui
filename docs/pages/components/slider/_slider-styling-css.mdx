Each `Slider` component's DOM node has a data attribute that you can target to style via CSS.

```css
/* outer container styles */
[data-interop-ui-slider] {
  /* */
}

/* track styles */
[data-interop-ui-slider-track] {
  /* */
}

/* range styles */
[data-interop-ui-slider-range] {
  /* */
}

/* thumb styles */
[data-interop-ui-slider-thumb] {
  /* */
}
```

Alternatively, you can pass class names to each component.

```css
.slider {
  /* */
}

.slider__track {
  /* */
}

.slider__range {
  /* */
}

.slider__thumb {
  /* */
}
```

```jsx
<Slider className="slider" defaultValue={[10, 30]}>
  <Slider.Track className="slider__track">
    <Slider.Range className="slider__range" />
  </Slider.Track>
  <Slider.Thumb className="slider__thumb" />
  <Slider.Thumb className="slider__thumb" />
</Slider>
```
