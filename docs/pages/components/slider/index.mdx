## Installation

Install the component from your command line:

```bash
npm install @interop-ui/react-slider
```

Then import the component in your project:

```js
import { Slider } from '@interop-ui/react-slider';
```

## Basic Usage

The most basic composition of a `Slider` component is as follows:

```jsx
// live
<Slider defaultValue={10}>
  <Slider.Track>
    <Slider.Range />
  </Slider.Track>
  <Slider.Thumb />
</Slider>
```

## Styling

All `Slider` components are unstyled by default. In most cases, each component renders a single DOM node which can be styled however you'd style a JSX component. In the event that a `name` prop is provided to `Slider`, it will also render a visually hidden input for each thumb that will be used to pass values into a form.
